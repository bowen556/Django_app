The first step as a superadmin user is to create different user groups and assign yourself or any new users to one of the groups to get access to the STS web application.
As a Super-admin user, you have to basically create two groups from the ‘Groups’ section of the admin page.
  • One group is called ‘admin’ for admin users that grants all the available permissions to make changes to the STS web application.
  • Another group is called ‘users’ for staff users with limited permission to only view the ‘Regions’ and ‘All Locations’ page.
Name of the user groups are case-sensitive so ensure that you use ‘admin’ and ‘user’ in the admin page.
  • Navigate to the ‘Users’ section of the admin page and then add new users by providing their username, password and personal information.
  • It is important to assign the users to one of the available groups to enable access to STS web application.
Though you have an option to grant individual permissions for each user, the permissions granted for each group supersedes the individual permissions. Therefore, even if you create new users by mapping them to ‘admin’ group but by restricting their individual permissions, they will still have access to all permissions granted for the admin group.
  • The authorized users who have attempted to login with incorrect password thrice will be in the blacklist category.
  • You have to go to the ‘Blacklists’ in the admin page and select the username you would like to provide access. Then, click ‘delete selected blacklists’ from the dropdown action to perform this change.
  • You have to select ‘Entrys’ section in the admin page and then click ‘add entry’ to save STS process data for a new location.
  • Follow the help text menu carefully and fill each of the field values in the mentioned format to view this information in the ‘viewlocations’ page without any issues.
You can verify the special characters entered for the problematic location from your updated location list. Please remove any copy pasted data fields with special characters and manually replace it with the standard characters supported in the QWERTY keyboard. This will facilitate to generate the PDF successfully from the ‘Viewlocations’ page.
  • You have to enter positive degree value for East and North direction whereas negative degree value for South and West direction.
  • Enter the rounded-up degrees, minutes and seconds without any decimals for both latitude and longitude.
  • You do not have to enter any values in the final STS Latitude and STS Longitude fields because the decimal coordinates is automatically calculated using the below formula and stored in the database.
    • Latitude/Longitude decimal= degrees + (min/60) + (sec/3600)
